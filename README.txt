Instruction for Programming Assignment #1 Closest pair of points

The program is written in Java program language. 
There are 4 Classes and 3 test txt files in the folder.
When running the program, make sure all test files and Java Class files are under the same folder.

4 Classes:

1.ClosestPointMain.java - MAIN entry for the whole program
2.Point			- Point class construct integers pair by pair
3.LoadData		- Read and parse txt data to Points type and stored in array list data structure.
4.ClosetPoint		- Divide and conquer Algorithm idea compute the closest distance between points.
